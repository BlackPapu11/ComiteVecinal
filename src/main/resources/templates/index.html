¿Quienes somos?

Somos una organización encargada y comprometida con la sociedad para canalizar las peticiones de las necesidades que tengan en sus colonia o localidades, por medio de esta aplicación se pueden formar comités vecinales, los cuales pueden reportar alguna necesidad o problema comunitario, y estás peticiones son canalizadas a la brevedad para ser atendidas por un enlace de cada municipio, los cuales se encargaran de ponerse en contacto con las personas indicadas para poder atender el problema lo más rapido posible.

Así funciona...

Existen comités vecinales, de los cuales los integrantes están en el deber de notificar al presidente de cada comité, para que este mismo realice un reporte de las incidencias y se atienda lo mas rápido posible, así mismo se puede realizar el pago desde la plataforma web y agilizar el procedimiento

Existen enlaces en cada municipio, los cuales cumplen la funcionalidad de canalizar las peticiones de las distintas incidencias con las personas indicadas, y así atender cada una de las peticiones de una manera eficiente sin perder el contacto con el presidente de cada comité.

Existe un administrador, el cual se encarga de regular la funcionalidad del siste, para que solo las personas que estén capacitadas tengan el acceso a el sistema y asi todo funcione de una manera correcta.
